# Connecting Claude.ai

## Steps

1. Open Claude.ai → Settings → Connectors
2. Click "Add custom connector"
3. Paste the URL: `https://mcp.nodaro.ai/mcp`
4. Click Add → you'll be redirected to Nodaro's sign-in page
5. Sign in with your Nodaro account
6. Review the consent screen and click Allow
7. Back in Claude.ai, the connector shows "Connected"
8. In a conversation, ask Claude to use a Nodaro tool: "Generate an image of a knight."
