# Connecting Goose

## Steps

1. Open Goose Desktop (or `goose configure` for the CLI)
2. Settings → Extensions → "Add custom extension"
3. Choose "Streamable HTTP" as the transport
4. Paste the URL: `https://mcp.nodaro.ai/mcp`
5. Save → Goose opens your browser to Nodaro's sign-in page
6. Sign in with your Nodaro account
7. Review the consent screen and click Allow
8. Back in Goose, the extension shows as enabled and the tools appear in the picker
9. Ask: "Generate an image of a knight using Nodaro."
