# Connecting Cursor

## Steps

1. Open Cursor → Settings (Cmd+,) → MCP
2. Click "Add new MCP server"
3. Choose "HTTP / Streamable HTTP" transport
4. Paste the URL: `https://mcp.nodaro.ai/mcp`
5. Save → Cursor opens your browser to Nodaro's sign-in page
6. Sign in with your Nodaro account
7. Review the consent screen and click Allow
8. Back in Cursor, the server shows as connected with available tools
9. Open the Composer / chat panel and ask: "Generate an image of a knight using Nodaro."
