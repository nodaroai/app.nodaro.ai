# Connecting Cline

## Steps

1. Open VS Code with the Cline extension installed
2. Click the Cline icon in the activity bar → Settings (gear icon)
3. Open the "MCP Servers" tab → "Add Remote MCP Server"
4. Paste the URL: `https://mcp.nodaro.ai/mcp`
5. Save → Cline opens your browser to Nodaro's sign-in page
6. Sign in with your Nodaro account
7. Review the consent screen and click Allow
8. Back in VS Code, the Cline panel shows the server as connected
9. In the Cline chat, ask: "Generate an image of a knight using the Nodaro MCP."
