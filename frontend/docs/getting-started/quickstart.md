# Quick Start

## 1. Create a Project

Navigate to `/projects` and click **New Project**. Give it a name and optional description.

## 2. Create a Workflow

Inside your project, go to the **Workflows** tab and click **New Workflow**. You can also organize workflows into folders.

## 3. Build Your Workflow

In the editor, add nodes from the left panel (desktop) or the + button (mobile):

- **Text Prompt** - Enter your story or concept text
- **Generate Script** - AI generates a structured script from your prompt
- **Generate Image** - AI generates images from text descriptions
- **Image to Video** - Convert images to video clips
- **Combine Videos** - Merge multiple clips into a final video

Connect nodes by dragging from output handles (right) to input handles (left).

## 4. Run

Click **Run** to execute the workflow. The system validates your graph first, showing any errors or warnings.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+D` / `Cmd+D` | Duplicate selected node |
| `Delete` | Delete selected node |
| `Escape` | Deselect / close panels |
| Right-click node | Context menu (duplicate, delete) |
